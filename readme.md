🎬 BotFlix

O BotFlix é um assistente interativo que ajuda você a encontrar filmes com base no seu humor ou preferência.
Basta digitar como você está se sentindo ou o tipo de filme que deseja assistir, e o BotFlix recomenda um título automaticamente.

🚀 Funcionalidades

Pesquisa de filmes a partir de um texto livre (ex: "quero algo engraçado", "estou com vontade de ver ação").

Integração com n8n + The Movie Database (TMDb API) para buscar os resultados.

Exibição de:

Pôster do filme

Título

Descrição

Nota média (⭐)

🛠️ Tecnologias Utilizadas

HTML5 para a estrutura da aplicação

CSS3 para estilização e layout responsivo

JavaScript (Vanilla JS) para interação e consumo da API

n8n para intermediar requisições entre a aplicação e o TMDb

TMDb API como fonte de dados de filmes
